Foobar
======

The Foobar provides the :ref:`foo` and :ref:`bar` functions.

.. _foo:

Foo
---

The ``foo`` function returns ``'foo'``.

.. code:: pycon

   >>> import foobar
   >>> foobar.foo()
   'foo'

.. _bar:

Bar
---

The ``bar`` function returns ``'bar'``.

.. code:: pycon

   >>> import foobar
   >>> foobar.bar()
   'bar'
